Udemy Clone Website

A responsive Udemy-inspired website built using pure HTML and CSS. This project recreates the basic layout and design of an online learning platform, including course sections, navigation bar, banners, and footer. It was created to practice front-end web development skills and improve understanding of website structure and styling.

Features
Responsive design
Modern UI layout
Navigation bar
Course cards section
Banner / Hero section
Footer section
Clean and simple design using HTML & CSS only
Technologies Used
HTML5
CSS3
Purpose

This project was built for learning and practice purposes to improve my front-end development skills by recreating a popular e-learning platform design.
